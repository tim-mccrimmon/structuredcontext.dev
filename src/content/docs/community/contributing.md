## Contributing to SCS

We welcome contributions to the Structured Context Specification!

:::tip[This is Spec Work]
SCS is v0.3 and actively evolving. If you're struggling with context management - or if you disagree with our framing - that's exactly the contribution we need.
:::

## How to Contribute

### Report Issues

- [GitHub Issues](https://github.com/structuredcontext/spec/issues)
- Bug reports
- Feature requests
- Spec clarifications

### Join Discussions

- [GitHub Discussions](https://github.com/structuredcontext/discussions)
- Share use cases
- Propose improvements
- Ask questions

### Contribute Code

- CLI tooling
- Validators
- Examples
- Documentation

### Contribute Bundles

- Standards bundles (HIPAA, SOC2, etc.)
- Example bundles
- Templates
