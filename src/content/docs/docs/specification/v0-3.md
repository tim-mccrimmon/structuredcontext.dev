## SCS v0.3 (Current)

The current version of the Structured Context Specification.

:::note[Active Development]
SCS v0.3 is actively evolving. This is spec work — contributions and feedback welcome.
:::

## What's in v0.3

- Five bundle types (Meta, Standards, Domain, Concern, Project)
- Three SCD tiers (meta, standards, project)
- Bundle import resolution
- Validation framework (loose/strict modes)
- Git-native versioning
- Provenance tracking

## Version history

- **v0.3** (Current) — Bundle type formalization
- **v0.2** — Import resolution
- **v0.1** — Initial specification
