## CLI Reference

Complete command reference for the SCS CLI.

## Installation

```bash
# npm
npm install -g @structuredcontext/cli

# pip
pip install scs-cli
```

## Commands

### Project Management

```bash
scs new <project-name>      # Create new project
scs init                    # Initialize in existing project
```

### Bundle Operations

```bash
scs bundle create           # Create new bundle
scs bundle version          # Version and lock bundle
scs validate               # Validate bundle/SCD
```
