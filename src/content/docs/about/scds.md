SCDs are the fundamental units of structured context in SCS.

## What is an SCD?

An SCD is a YAML file containing structured context organized by tier:

- **Meta Tier**: SCS specification language
- **Standards Tier**: Compliance requirements
- **Project Tier**: Company/project-specific context
